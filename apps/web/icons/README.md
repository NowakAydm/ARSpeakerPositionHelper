# Icons Directory

This directory contains PWA icons in various sizes for different devices and use cases.

## Icon Sizes

- 72x72: Basic icon for older devices
- 96x96: Standard mobile icon
- 128x128: Chrome Web Store
- 144x144: Windows tiles
- 152x152: iPad home screen
- 192x192: Android home screen
- 384x384: Large displays
- 512x512: Splash screen and high-resolution displays

## Icon Guidelines

Icons should:
- Be simple and recognizable at small sizes
- Use the app's primary colors
- Include padding for the "maskable" format
- Be optimized for file size

To generate icons from a source image, use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Adobe Photoshop or GIMP with batch processing

## Maskable Icons

All icons include the "maskable" purpose to work with adaptive icons on Android and other platforms that may crop the icon into different shapes.
